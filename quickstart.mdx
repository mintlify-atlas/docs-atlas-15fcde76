This guide will walk you through the core TaskFlow API workflow. By the end, you'll have a working board with tasks.

<Note>
  Before starting, ensure the API is running locally at `http://localhost:5000` or your configured port. See the [Installation Guide](/installation) for setup instructions.
</Note>

## Step 1: Register a User

Create your first user account. Registration automatically creates a personal board with default columns.

<CodeGroup>
```bash cURL
curl -X POST http://localhost:5000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "Joel",
    "email": "joel@test.com",
    "password": "123456"
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost:5000/api/auth/register', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    username: 'Joel',
    email: 'joel@test.com',
    password: '123456'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
  'http://localhost:5000/api/auth/register',
  json={
    'username': 'Joel',
    'email': 'joel@test.com',
    'password': '123456'
  }
)

print(response.json())
```
</CodeGroup>

**Response:**
```json
{
  "message": "User registered successfully"
}
```

<Info>
  On registration, TaskFlow automatically:
  - Creates a personal board named "Joel's Board"
  - Assigns you as Owner of that board
  - Creates three default columns: "To Do", "In Progress", and "Done"
</Info>

## Step 2: Login and Get Token

Authenticate to receive a JWT token. This token must be included in all subsequent requests.

<CodeGroup>
```bash cURL
curl -X POST http://localhost:5000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "joel@test.com",
    "password": "123456"
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost:5000/api/auth/login', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    email: 'joel@test.com',
    password: '123456'
  })
});

const { token } = await response.json();
console.log(token);
```

```python Python
import requests

response = requests.post(
  'http://localhost:5000/api/auth/login',
  json={
    'email': 'joel@test.com',
    'password': '123456'
  }
)

token = response.json()['token']
print(token)
```
</CodeGroup>

**Response:**
```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}
```

<Warning>
  Store this token securely. It expires after 60 minutes by default (configurable in `appsettings.json`). Include it in the `Authorization` header as `Bearer <token>` for all protected endpoints.
</Warning>

## Step 3: Get Your Boards

Retrieve all boards where you're a member. Since registration created a personal board, you'll see it here.

<CodeGroup>
```bash cURL
curl -X GET http://localhost:5000/api/board/my \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

```javascript JavaScript
const response = await fetch('http://localhost:5000/api/board/my', {
  headers: {
    'Authorization': `Bearer ${token}`
  }
});

const boards = await response.json();
console.log(boards);
```

```python Python
import requests

response = requests.get(
  'http://localhost:5000/api/board/my',
  headers={'Authorization': f'Bearer {token}'}
)

boards = response.json()
print(boards)
```
</CodeGroup>

**Response:**
```json
[
  {
    "id": 1,
    "name": "Joel's Board",
    "joinCode": "ABC123"
  }
]
```

## Step 4: View Board Details

Get the complete board structure including columns and tasks.

<CodeGroup>
```bash cURL
curl -X GET http://localhost:5000/api/board/1 \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

```javascript JavaScript
const boardId = 1;
const response = await fetch(`http://localhost:5000/api/board/${boardId}`, {
  headers: {
    'Authorization': `Bearer ${token}`
  }
});

const board = await response.json();
console.log(board);
```

```python Python
import requests

board_id = 1
response = requests.get(
  f'http://localhost:5000/api/board/{board_id}',
  headers={'Authorization': f'Bearer {token}'}
)

</CodeGroup>

    {
      "id": 3,
      "name": "Done",
      "position": 2,

  ]

```

## Step 5: Create a Task

Add your first task to the "To Do" column (column ID 1).

<CodeGroup>
```bash cURL
curl -X POST http://localhost:5000/api/task \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -H "Content-Type: application/json" \
  -d '{
    "columnId": 1,
    "title": "Setup development environment",
    "description": "Install .NET 8, MySQL, and configure the database connection"
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost:5000/api/task', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    columnId: 1,
    title: 'Setup development environment',
    description: 'Install .NET 8, MySQL, and configure the database connection'
  })
});

const task = await response.json();
console.log(task);
```

```python Python
import requests

response = requests.post(
  'http://localhost:5000/api/task',
  headers={
    'Authorization': f'Bearer {token}',
    'Content-Type': 'application/json'
  },
  json={
    'columnId': 1,
    'title': 'Setup development environment',
    'description': 'Install .NET 8, MySQL, and configure the database connection'
  }
)

task = response.json()
print(task)
```
</CodeGroup>

**Response:**
```json
{
  "id": 1,
  "title": "Setup development environment",
  "description": "Install .NET 8, MySQL, and configure the database connection",
  "position": 0,
  "columnId": 1
}
```

<Info>
  The `position` field is automatically calculated. New tasks are added to the end of the column.
</Info>

## Step 6: Move a Task

Move the task from "To Do" (column 1) to "In Progress" (column 2).

<CodeGroup>
```bash cURL
curl -X PUT http://localhost:5000/api/task/1/move \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -H "Content-Type: application/json" \
  -d '{
    "newColumnId": 2,
    "newPosition": 0
  }'
```

```javascript JavaScript
const taskId = 1;
const response = await fetch(`http://localhost:5000/api/task/${taskId}/move`, {
  method: 'PUT',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    newColumnId: 2,
    newPosition: 0
  })
});

const movedTask = await response.json();
console.log(movedTask);
```

```python Python
import requests

task_id = 1
response = requests.put(
  f'http://localhost:5000/api/task/{task_id}/move',
  headers={
    'Authorization': f'Bearer {token}',
    'Content-Type': 'application/json'
  },
  json={
    'newColumnId': 2,
    'newPosition': 0
  }
)

moved_task = response.json()
print(moved_task)
```
</CodeGroup>

<Info>
  TaskFlow automatically:
  - Removes the task from its old column and adjusts positions
  - Inserts it into the new column at the specified position
  - Shifts other tasks to maintain correct ordering
</Info>

## Step 7: Create Additional Boards

Create more boards for different projects.

<CodeGroup>
```bash cURL
curl -X POST http://localhost:5000/api/board \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Q1 Product Launch"
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost:5000/api/board', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Q1 Product Launch'
  })
});

const newBoard = await response.json();
console.log(newBoard);
```

```python Python
import requests

response = requests.post(
  'http://localhost:5000/api/board',
  headers={
    'Authorization': f'Bearer {token}',
    'Content-Type': 'application/json'
  },
  json={'name': 'Q1 Product Launch'}
)

new_board = response.json()
print(new_board)
```
</CodeGroup>

**Response:**
```json
{
  "id": 2,
  "name": "Q1 Product Launch",
  "joinCode": "XY7K9P"
}
```

<Note>
  Each board gets a unique 6-character `joinCode`. Share this code with team members so they can join as Members using the `/api/board/join` endpoint.
</Note>
