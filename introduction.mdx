## Welcome to TaskFlow API

TaskFlow is a comprehensive RESTful API for building Trello-style task management applications. Built with modern ASP.NET Core technology, it provides everything you need to create collaborative Kanban boards with sophisticated permission management.

## Technology Stack

TaskFlow is built with modern, production-ready technologies:

- **ASP.NET Core 8** - High-performance web framework
- **Entity Framework Core** - Object-relational mapper with LINQ support
- **MySQL** - Reliable relational database with Pomelo provider
- **JWT Authentication** - Industry-standard token-based auth
- **Swagger/OpenAPI** - Interactive API documentation

## How It Works

<Steps>
  <Step title="Register & Authenticate">
    Users register with email and password. On registration, a personal board is automatically created with default columns (To Do, In Progress, Done). Login returns a JWT token for subsequent requests.
  </Step>
  
  <Step title="Create & Share Boards">
    Create multiple boards for different projects. Each board gets a unique join code that others can use to join as members. The creator becomes the Owner with full control.
  </Step>
  
  <Step title="Organize with Columns">
    Add custom columns to organize your workflow. Owners and Admins can create, rename, reorder, and delete columns. Position management is handled automatically.
  </Step>
  
  <Step title="Manage Tasks">
    All board members can create tasks, add descriptions, set due dates, and move tasks between columns. Tasks maintain their position and can be assigned to specific users.
  </Step>
</Steps>

## Role-Based Permissions

TaskFlow implements a three-tier permission system:

| Action | Owner | Admin | Member |
| --- | --- | --- | --- |
| Create/Delete Board | ✓ | ✗ | ✗ |
| Create/Delete Column | ✓ | ✓ | ✗ |
| Create/Move Tasks | ✓ | ✓ | ✓ |
| View Board | ✓ | ✓ | ✓ |

<Note>
  Permissions are validated on every request. Only board members can access board data, ensuring complete isolation between boards.
</Note>
